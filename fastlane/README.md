# Installation

Make sure you have the latest version of the Xcode command line tools installed:

```
xcode-select --install
```

Install _fastlane_ using
```
[sudo] gem install fastlane -NV
```
or alternatively using `brew cask install fastlane`

# Available Actions
### rename
```
fastlane rename
```
Rename project template

Use this lane when you're creating a new project to rename it from project template to desired name.
#### Options
 * **`name`** (Optional): New name of project. If not provided user is prompted to write one.
